# ⚛️ React + TypeScript Fundamentals

A React + TypeScript starter project illustrating typed components and state management.

---

## 🔧 Prerequisites

- Node.js 16+ installed
- npm/yarn/pnpm installed

---

## 🚀 Quick start (local)

```bash
cd reactjs/01-fundamentos-reactjs-ts
npm install
npm run dev
```
