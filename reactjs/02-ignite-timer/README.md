# ⏱️ Timer App

A React timer application demonstrating countdown behavior, controls, and component state.

---

## 🔧 Prerequisites

- Node.js 16+ installed
- npm/yarn/pnpm installed

```bash
cd reactjs/02-ignite-timer
npm install
npm run dev
```
