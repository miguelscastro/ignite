# Ignite
this repository contains all projects and APIs developed either during classes or as module aprovation exams, which are marked as submodules and deployed for better visualization if React projects.

## React
If you want to run the project locally, after cloning, go into it's main folder and procede with Setup.

### Setup
inside the project folder, install dependencies:
```
npm install
```

Now just use:
```
npm run dev
```

You should find the application running at http://localhost:3000

## Java Spring Boot
For spring applications, after cloning the API you want to test procede with Setup.

### Setup
inside the project folder, install dependencies:
```
mvn install
```
Then procede to the main Class marked with the @SpringBootApplication annotation and press F5 to run (requires Java RedHat Extension Pack if using VsCode)

You should find the application running at http://localhost:8080 and the PostgreSQL database listening at PORT:5432
