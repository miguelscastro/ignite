# 🛍️ Ignite Shop

An e-commerce storefront demo built with Next.js and server-rendered pages.

---

## 🔧 Prerequisites

- Node.js 16+ installed
- pnpm/npm/yarn installed

---

## 🚀 Quick start (local)

```bash
cd nextjs/01-ignite-shop
pnpm install
pnpm dev
```

---

## 🧪 Tests

- Use `pnpm test` if present in package scripts.

---

## 🔁 Features (summary)

- Product listing pages
- Shopping cart UI
- Demo checkout flow

---

## 💡 Notes

- Check `next.config.ts` for custom runtime settings.

---

Happy coding! ✅
