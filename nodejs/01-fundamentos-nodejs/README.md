# Node.js Fundamentos

A Node.js fundamentals project with basic exercises and HTTP server examples.

---

## 🔧 Prerequisites

- Node.js 16+ installed

---

## 🚀 Quick start (local)

```bash
cd nodejs/01-fundamentos-nodejs
npm install
npm run dev
```

## 🔁 Features (summary)

- HTTP server usage
