# 🌐 Gestão de Vagas Web

A web frontend application using Spring Boot with server-rendered pages for vacancy management.

---

## 🔧 Prerequisites

- Java 11+ installed
- Maven installed (or use `./mvnw`)
- Backend API running (default: http://localhost:8080)

---

## 🚀 Quick start (local)

```bash
cd java/05-gestao-vagas-web
./mvnw spring-boot:run
```

---

## 🧪 Tests

- Run `./mvnw test` for tests.

---

## 🔁 Features (summary)

- UI for job vacancies
- Integration with backend API
- Configuration via `application.properties`

---

## 💡 Notes

- Update `src/main/resources/application.properties` if backend URL differs.

---

Happy coding! ✅
